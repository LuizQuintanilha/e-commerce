Projeto E-commerce

Administrador deverá ser capaz de cadastrar produtos
Produtos devem possuir:
1 - Nome 
2 - Categoria
3 - Código de identificação único
4 - Preço varejo
5 - Preço atacado
6 - Disponibilidade do produto
7 - Imagem


Projeto criado com:
ruby 3.0.0p0
Rails 7.0.7.2
Rubocop 1.52.0
I18n
